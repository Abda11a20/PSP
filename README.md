# PSP - Security Awareness Platform

## 📌 Overview

PSP is a web application built with Laravel that helps companies simulate phishing attacks and analyze user behavior to improve security awareness.

---

## 🚀 Features

* User Authentication (Login, Register, OTP, 2FA)
* Campaign Management
* Phishing Email Simulation
* User Interaction Tracking
* AI Analysis
* Payment System
* Admin & Client Dashboards

---

## 🛠️ Technologies

* PHP (Laravel)
* MySQL
* Blade Templates
* JavaScript / CSS

---

## 📂 Project Structure

* **Models** → Handle database
* **Controllers** → Business logic
* **Views** → UI
* **Services** → Core functionalities

---

## ⚙️ Installation

```bash
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
```
